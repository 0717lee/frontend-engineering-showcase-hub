# Frontend Engineering Showcase Hub

> This is the engineering narrative layer for the frontend engineering system.

## What This Is

A static site that explains the frontend engineering system architecture, deployment strategies, and project management decisions.

## What This Is NOT

- ❌ Not an engineering control panel
- ❌ Not a management system
- ❌ Not connected to any backend APIs
- ❌ Not involved in engineering operations

## Engineering Role

| Project | Platform | Role |
|---------|----------|------|
| Engineering Control Panel | Self-hosted | Engineering operations & monitoring |
| **Showcase Hub (this)** | **Vercel** | **Engineering narrative & explanation** |
| CollabBoard | Vercel + Cloudflare | Product delivery |
| Lumina Flow | Vercel (planned) | Engineering validation |

## Tech Stack

- React 19 + TypeScript
- Vite
- React Router
- Vanilla CSS
- Bilingual (EN/ZH)

## Local Development
